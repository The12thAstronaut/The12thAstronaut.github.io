Howdy, we are the Texas A&M NASA Spacesuit User Interface Technologies for Students (SUITS) competition team. The NASA SUITS competition was started in 2017 to allow students to help contribute to the design process of the Joint Augmented Reality Visual Informatics System (JARVIS). JARVIS will ultimately be a heads-in display for the Exploration Extravehicular Mobility Unit (xEMU) the new spacesuit to be implemented in the Artemis missions. NASA SUITS challenges students to create a user interface on a heads-up display incorporating Extra-Vehicular Activity (EVA) tasks, navigation, communication, and the ability to sort through telemetry.
